Options:
  -V, --version             output the version number
  --no-color                Disable colored output
  -h, --help                display help for command

Commands:
  init                      Generate a .migrationpilotrc.yml config file
  list-rules [options]      List all available safety rules
  detect [dir]              Auto-detect migration framework and suggest
                            configuration
  watch [options] <dir>     Watch migration files and re-analyze on change
  hook <action>             Install or uninstall git pre-commit hook
  plan [options] <file>     Show a visual execution plan for a migration file
  analyze [options] [file]  Analyze a SQL migration file for safety
  check [options] <dir>     Check all migration files in a directory
  help [command]            display help for command
